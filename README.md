**!! wip !!** i got really bored and made this. practising HTML with the help of LLM like Claude and ChatGPT.

**the goal:** build something to help myself whenever i go "what task was I doing before I got distracted/interrupted?".
**the idea:** minimalist clock, list and prompt box. once clock starts countdown, only 1 task viewable. once session ends, back to homepage, can select new task.
**UVP:** during focus session, there's a prompter box. i can "braindump" any random thought i have here, it will be filed away but out of sight - out of mind (at least until end of session)

inspiration from windows 11 clock app + google task + my visual countdown clock.
